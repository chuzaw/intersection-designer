# 三、道路进口几何生成

假设：


## 1. 方向向量

角度转向量：

公式：

```
dx = cos(angle)

dy = sin(angle)
```

90度：

得到：

```
dx=0
dy=1
```

即：

向北。

```
    |
```

```
(0,0)
```

# 五、多个进口旋转

关键：

不要分别写北东南西。

统一：

```
旋转矩阵
```

公式：

点(x,y)

旋转θ：

```
x'=x*cosθ-y*sinθ

y'=x*sinθ+y*cosθ
```

这样：

同一个道路模板：

```
|
|
|

旋转：

90°

↓

---

270°

↓

---
```

即可。
